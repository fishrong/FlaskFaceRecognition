# 基于web的在线人脸识别系统
## 所涉及的python库
* Flask
* face_recognition

## 实现功能
* 可以拖拽文件或者在系统的文件浏览中选择文件进行人脸识别
* 可以对未知的人脸进行训练命名
* 支持调用摄像头实时识别
* 适配移动端与pc端
